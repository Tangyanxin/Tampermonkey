# bilibili订阅+

## 描述
如图所示，在bilibili导航栏添加了一个订阅的按钮，可以在bilibili任何页面快速访问正在追的番剧

**安装了v0.3.\*了的同学请卸载该版本重新安装v0.4.\*,否则导航栏会出现两个订阅，谢谢**

## 特色
1. bilibili所有页面快速访问订阅番剧
2. 自定义添加位置
